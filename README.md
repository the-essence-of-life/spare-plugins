# spare-starter

## Quick Start

```shell
git clone https://github.com/the-essence-of-life/starter ~/.config/nvim/
```

## Migration

```shell
mkdir -p ~/.config/nvim/lua/config/
```

## Uninstall

```shell
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
rm -rf ~/.config/nvim
```
